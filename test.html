테스트 명세서 정리

테스트 : 단위 테스트, 3자 테스트, 통합 테스트, 사용자 테스트 (베타 테스트)
단위 테스트 - 기능 별로 작성, 개발 과정에서 기능이 완성될 때 마다 개발자가 직접 테스트
3자 테스트 - 개발자 본인이 아닌 동료 개발자 또는 다른 테스터에 의해 진행

  1. 통합 테스트

    2. 시나리오 기반으로 진행
    앱의 설치부터 로그인, 화면 이동, 서버와의 데이터 통신 전반 점검

  2. 디바이스 테스트

    3. 타겟 디바이스에 앱 설치 후 테스트 진행 ( 앱 설치와 주요 기능 위주로 )
    디바이스 확보가 어려운 경우에는 Genymotion 서비스 추천

  3. 사용자 베타 테스트

    4. 실제 사용자 중 일부에게 빌드된 앱을 발송하여 베타 테스트 진행
    이 테스트를 통해 사용자의 의견을 취합하고 심각한 오류나 버그 수정
